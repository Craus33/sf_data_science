### Описание проекта    
Угадать загаданное компьютером число за минимальное число попыток.

:arrow_up:[к оглавлению](https://github.com/Craus33/sf_data_science/tree/main/Project_0#%D0%BE%D0%B3%D0%BB%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5)


### Какой кейс решаем?    
Нужно написать программу, которая угадывает число за минимальное число попыток

**Условия соревнования:**  
- Компьютер загадывает целое число от 0 до 100, и нам его нужно угадать. Под «угадать», подразумевается «написать программу, которая угадывает число».
- Алгоритм учитывает информацию о том, больше ли случайное число или меньше нужного нам.

**Метрика качества**     
Результаты оцениваются по среднему количеству попыток при 1000 повторений

**Что практикуем**     
Учимся писать хороший код на python


### Краткая информация о данных
Число загадывается рандомно в диапозоне от 0 до 100, ограничение по количеству попыток - 1000
  
:arrow_up:[к оглавлению](https://github.com/Craus33/sf_data_science/tree/main/Project_0#%D0%BE%D0%B3%D0%BB%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5)


### Этапы работы над проектом  
1. Программа рандомно выбирает число от 0 до 100, пользователь пробует угадать, вводя числа с клавиатуры 
2. Модифицируем код, автоматизируеи перебор вариантов из диапазона 0-100
3. Оптимизируем код, чтобысократить количество попыток до 20

:arrow_up:[к оглавлению](https://github.com/Craus33/sf_data_science/tree/main/Project_0#%D0%BE%D0%B3%D0%BB%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5)


### Результаты:  
1. Пользователь в среднем угадывает число за 6 попыток
2. Программа с неоптимальным кодом угадывает за 101 попытку
3. Программа с оптимальным кодом угадывает число за 8 попыток
